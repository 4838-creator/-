# 業務日報アプリ セットアップ手順

このアプリは、**1日分の業務内容をリスト形式で入力**し、Googleスプレッドシートに一括保存するWebアプリケーションです。

## 📁 ファイル構成

- `index.html`: アプリの画面（リスト形式）
- `style.css`: デザイン
- `script.js`: アプリの動作ロジック（**設定変更が必要です**）
- `gas_code.js`: Google Apps Script用のコード（コピペ用）

## 🚀 セットアップ（Googleスプレッドシート連携）

ご自身のGoogleアカウントで、以下の手順を行ってください。

### 1. スプレッドシートの準備
1. Googleスプレッドシートを新規作成します。
2. 1行目にヘッダーを入力しておくと見やすいです（例：A1「日時」、B1「対象時間帯」、C1「業務内容」）。

### 2. Google Apps Script (GAS) の設定
1. スプレッドシートのメニューから **「拡張機能」** > **「Apps Script」** をクリックします。
2. スクリプトエディタが開くので、元々あるコード（`function myFunction...`）をすべて削除します。
3. このフォルダにある **`gas_code.js` の中身をすべてコピーし、スクリプトエディタに貼り付けます。**
   ※以前のバージョンをお使いの場合も、コードを**全置換**してください。
4. フロッピーアイコン💾をクリックして保存します。

### 3. Webアプリとしてデプロイ（重要）
**※コードを変更した場合は、必ず「新しいデプロイ」を行ってください！**

1. 画面右上の **「デプロイ」** ボタン > **「新しいデプロイ」** を選択します。
2. 左側の「種類の選択」歯車アイコン⚙️をクリックし、 **「ウェブアプリ」** を選択します。
3. 設定項目を以下のようにします：
    - **説明**: 任意（例: 日報API ver2）
    - **次のユーザーとして実行**: **自分** (Me)
    - **アクセスできるユーザー**: **全員** (Anyone)
4. **「デプロイ」** ボタンをクリックします。
5. 新しく発行された **「ウェブアプリのURL」** をコピーしてください。
   （以前のURLと変わる場合があります）

### 4. アプリへのURL設定
1. このフォルダの `script.js` をテキストエディタで開きます。
2. 5行目付近にある `GAS_API_URL` の変数に、先ほどコピーしたURLを貼り付けます。

```javascript
const GAS_API_URL = "https://script.google.com/macros/s/xxxxxxxxxxxxxxxxx/exec";
```

3. 上書き保存します。

## ✅ 動作確認
1. `index.html` をブラウザ（Chromeなど）で開きます。
2. 今日の日付が選択され、9:00〜22:00頃までのリストが表示されていることを確認します。
3. いくつかの時間帯に入力して「一括送信する」を押します。
4. スプレッドシートに入力した数だけ行が追加されていれば成功です！
